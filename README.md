# Network Tool Plus

一个 Minecraft NeoForge 1.21.1 模组，为 AE2 **网络工具** 添加自动收集功能。

## 功能

- **收集模式** — 潜行 + 右键切换，激活时工具显示附魔光效
- **自动收卡** — 开启后自动将背包中的 AE2 升级卡吸入工具，工具放在背包里也生效
- **21 格升级槽** — 从原版 9 格扩展至 21 格（7×3 网格）
- **自定义 GUI** — 替换 AE2 网络工具界面与工具箱面板纹理

## 依赖

- Minecraft **1.21.1**
- NeoForge **21.1.102+**
- Applied Energistics 2 (AE2) **19.0.4+**
- Curios (可选) **9.1.0+**
