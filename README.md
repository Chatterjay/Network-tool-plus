# Network Tool Plus

一个 Minecraft NeoForge 1.21.1 模组，为 AE2 **网络工具** 添加自动收集功能与工具箱增强。

## 功能

- **收集模式** — 潜行 + 右键切换，激活时工具显示附魔光效
- **自动收卡** — 开启后自动将背包中的 AE2 升级卡吸入工具，工具放在背包或 Curios 槽中均生效
- **21 格升级槽** — 从原版 9 格扩展至 21 格（7×3 网格）
- **工具箱定向填充** — 在机器 GUI 中，从工具箱 shift-click 升级卡时直接填入升级槽，不兼容时卡留原位，不会误入物品槽。终端中保留原行为
- **工具槽位锁定** — 打开任意 AE2 菜单时自动锁定背包中的网络工具，防止误移动

## 依赖

- Minecraft **1.21.1**
- NeoForge **21.1.102+**
- Applied Energistics 2 (AE2) **19.0.4+**
- Curios (可选) **9.1.0+**
